# CloneTracker

Система учёта клонов для скрининга мутантов A. niger.

## Быстрый старт

```bash
npm install
npm run dev
```

## Разработка с Claude Code

```bash
cd clone-tracker
claude
```

Claude Code прочитает `CLAUDE.md` и будет знать контекст проекта.

## Текущее состояние

MVP в одном файле `src/App.jsx`. Следующий шаг — декомпозиция на компоненты (структура описана в CLAUDE.md).
